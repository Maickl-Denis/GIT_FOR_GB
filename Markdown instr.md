## Ссылки

Чтобы добавить ненумерованые списки, необходимо пункты выделить звездочкой (*).
Например, вот так:
* элемент 1
* элемент 2
* элемент 3

Чтобы добавить нумерованые списки, необходимо просто пронумеровать.
Например, вот так:
1. элемент 1
2. элемент 2
3. элемент 3
