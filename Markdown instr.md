# Инструкция для работы с Mardown

## Выделение тукста

Чтобы выделить текст курсивом необходимо обрамитьб его звездачками (*). Например, *вот так*
    
Чтобы выделить текст курсивом необходимо обрамитьб его двойными звездачками (**).
Например, **Вот так**.
